# はじめに

個人サイトをはじめました。職務経歴書を兼ねた読み物として、案件の文脈と役職の遷移、
そこで考えたことをふんわりと残していきます。

## 構成

- Next.js (App Router) + TypeScript + Tailwind CSS
- 静的エクスポートして GitHub Pages にデプロイ
- ブログは `content/blog/*.mdx` を置くだけで自動公開
- 経歴は `content/career.json` に構造化（案件内の役職遷移を `roles[]` で表現）

```ts
type Role = {
  title: string;
  start: string; // YYYY-MM
  end: string | null;
};
```

## これから書くこと

採用担当・コミュニティ・知人に向けて、SNS や GitHub プロフィールでは伝わらない
案件文脈や学びを淡く綴っていきます。
