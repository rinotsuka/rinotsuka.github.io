このブログ本文の見た目は [locabo.net/about/](https://locabo.net/about/) のメインコンテンツに寄せています。
サイト全体のカワイイ系トーンは Header / Footer / カードラッピングで出しつつ、本文は装飾を抑えて読みやすさを優先しました。

## サンプル

このページは Markdown / MDX で表示されうる主要な要素をひととおり並べた表示確認用です。

### 段落とインライン要素

これは普通の段落です。長めの文章が続いた場合の行間や文字色を確認します。本文は `--color-ink-soft` を使い、淡めのトーンです。

**太字（strong）** と *斜体（em）*、~~取り消し線~~、それから `インラインコード` の見え方。
リンクは [外部リンクサンプル](https://example.com) のように下線つきで表示されます。

### 順序なしリスト

- 箇条書き 1
- 箇条書き 2
  - ネスト 2-1
  - ネスト 2-2
    - さらにネスト
- 箇条書き 3

### 順序つきリスト

1. はじめに
2. 本論
   1. 補足 A
   2. 補足 B
3. まとめ

### タスクリスト（GFM）

- [x] 完了したタスク
- [x] もう一つの完了
- [ ] 未完了のタスク

### 引用

> シンプルな引用ブロック。
> 複数行にわたる引用もそのまま流せます。

> 引用の中にも段落を入れられます。
>
> 二段落目もこのとおり。
