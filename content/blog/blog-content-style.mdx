このブログ本文の見た目は [locabo.net/about/](https://locabo.net/about/) のメインコンテンツに寄せています。
サイト全体のカワイイ系トーンは Header / Footer / カードラッピングで出しつつ、本文は装飾を抑えて読みやすさを優先しました。

## 採用したルール

- 白背景・サンセリフ
- 本文 max-width は 760px 前後
- 見出しは下線のみ（背景色やアイコンは置かない）
- 引用は左ボーダー＋淡色背景の標準的なスタイル
- リンクは下線つき、色はサイトのプライマリ（オレンジ）

### コードブロック

エンジニア向けなので、コードブロックは黒地で読みやすく:

```tsx
export function Greeting({ name }: { name: string }) {
  return <p>こんにちは、{name} さん。</p>;
}
```

### リストと表

| 項目         | 値              |
| ------------ | --------------- |
| max-width    | 760px           |
| 行間         | 1.85            |
| 字間         | 0.01em          |

1. 装飾はサイト全体で出す
2. 本文は読みやすさ優先
3. 例外として CTA や装飾的な囲みは個別に作る
